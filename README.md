# TechNova Solutions — Corporate Website

A complete, responsive multi-page business/IT company website built using only **HTML5** and **CSS3** — no JavaScript, no frameworks (no Bootstrap/Tailwind).

## Project Overview

TechNova Solutions is a fictional IT services company website created as a front-end development project. It demonstrates a full corporate site structure: home, about, services, portfolio, blog, and contact pages, all sharing one centralized stylesheet.

## Features

- Fully responsive design (breakpoints at 1200px, 992px, 768px, 576px)
- Sticky navigation bar with active-page highlighting
- CSS-only animated hero section with a network-node motif
- CSS Grid & Flexbox layouts throughout
- CSS-only testimonial slider (no JavaScript)
- Hover effects on cards, buttons, images, and navigation (pure CSS)
- CSS custom properties (variables) for colors, spacing, typography, shadows
- CSS animations: fade in, slide up, slide left, slide right, zoom in
- Semantic HTML5 (`header`, `nav`, `main`, `section`, `article`, `aside`, `footer`)
- SEO meta tags (title, description, keywords) on every page
- Accessible markup: alt text on all images, proper heading hierarchy, form labels
- `prefers-reduced-motion` support for accessibility

## Technologies Used

- HTML5 (semantic markup)
- CSS3 (Grid, Flexbox, custom properties, animations, transitions)
- Google Maps embed (iframe) on the Contact page
- Unsplash-hosted images for visual content

## Pages Summary

| Page | Key Sections |
|---|---|
| `index.html` | Hero, About preview, Features (6), Services preview (6), Stats, Why Choose Us, Testimonials (5), Team (4), CTA |
| `about.html` | Company story, Mission & Vision, Timeline (2018–2025), Values (6), Team, Achievements |
| `services.html` | 12 service cards, Pricing (3 tiers), FAQ (10) |
| `portfolio.html` | Category filters, 12 projects, 12-image gallery |
| `blog.html` | Featured article, 10 blog cards, sidebar (search, categories, recent posts, tags) |
| `contact.html` | Contact form, company info, map, 3 office locations |

## How to Run

No build tools or server required.

1. Download/unzip the project folder.
2. Open `index.html` directly in any modern web browser (Chrome, Firefox, Edge, Safari).
3. Navigate between pages using the top navigation bar.

## Notes

- All images are loaded from Unsplash via direct URLs and require an internet connection to display.
- The contact form is front-end only (no backend); submitting it will not send actual data anywhere.
- Replace the Google Maps `src` URL in `contact.html` with your own embed code if reusing this template  for a real business.
